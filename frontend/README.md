# Frontend - re-frame.social

This directory contains the Next.js 14 frontend application for the CBT Assistant.

## Quick Start

```bash
cd frontend
pnpm install
pnpm run dev
```

## Build

```bash
pnpm run build
pnpm run start
```

## Testing

```bash
pnpm run test
pnpm run test:watch
```

See the main [README.md](../README.md) for full project documentation.
